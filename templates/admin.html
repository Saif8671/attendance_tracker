❓ Why Rows Get Skipped

Empty Name
The Name column is blank for that row
Duplicate Username
Same HTNO or username already exists → auto-renamed with suffix
Wrong Column Names
Column header doesn't match any accepted name above
Completely Empty Row
Blank lines in the CSV are silently ignored
Wrong File Encoding
Save your file as "CSV UTF-8" from Excel (not just CSV)
Extra Spaces in Headers
e.g. " Name " with spaces — use Preview to check
